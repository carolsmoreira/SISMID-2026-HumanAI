# SISMID-2026

Materials for **SISMID 2026**.

## Overview

This repository contains course/workshop content and supporting files for SISMID 2026.

## Repository structure

Because this repository is primarily HTML with some R code, you can expect content such as:

- HTML pages and assets for rendered materials
- R scripts and/or notebooks used to generate analyses or figures

## Getting started

1. Clone the repository:
   ```bash
   git clone https://github.com/bleicham/SISMID-2026.git
   cd SISMID-2026
   ```
2. Open the HTML files directly in a browser, or serve the directory locally.
3. Run R scripts in an R environment (RStudio recommended).

## Requirements

- A modern web browser (for HTML content)
- [R](https://cran.r-project.org/) (for `.R` files)
- Optional: RStudio for easier workflow
